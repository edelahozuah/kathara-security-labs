# Práctica 2 - Laboratorios de Seguridad

## Escenario 1: Topología Básica

Escenario con 4 hosts (h1-h4) y 2 routers (r1, r2) conectados mediante 3 redes virtuales.

**Objetivo**: Configuración básica de routing y forwarding.

**Características**:
- h1 con GUI (LXQt + VNC) accesible por VPN WireGuard
- Scripts de gestión cross-platform

## Escenario 2: Escenario de Ataque

Escenario de seguridad con nodos atacante, víctima, NAT gateway y router.

**Objetivo**: Prácticas de análisis de tráfico y configuración de defensas.

**Características**:
- Nodo desktop con GUI
- Documentación extendida
