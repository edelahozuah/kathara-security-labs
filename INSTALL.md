# Instalacion de Kathara

Para evitar duplicar pasos (y que queden obsoletos), este repositorio no mantiene
instrucciones detalladas de instalacion de Kathara.

Usa siempre la guia oficial:

- https://github.com/KatharaFramework/Kathara/wiki/Installation-Guides

Referencias oficiales adicionales:

- https://www.kathara.org/
- https://github.com/KatharaFramework/Kathara

Despues de instalar, puedes validar tu entorno con:

```bash
kathara --version
kathara check
```
